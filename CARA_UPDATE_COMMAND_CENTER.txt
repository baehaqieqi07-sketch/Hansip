## Tes lokal

```bash
npm install
node --check index.js
npm start
```

## Setelah online

Coba command:

```txt
othelp
otping
otinfo
otstatus
```

Kalau `otstatus` ditolak, isi Owner User ID lewat Railway Variables:

```env
OWNER_USER_ID=ID_DISCORD_KAMU
```

atau isi lewat dashboard Permission Center.
